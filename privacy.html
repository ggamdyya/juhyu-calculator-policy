1. 개인정보 처리방침의 목적

본 개인정보 처리방침은 개발자 "깜디야"가 제공하는 모바일 애플리케이션 ("주휴수당 계산기" 및 "4대보험 계산기", 이하 "본 앱")을 이용함에 있어 사용자의 개인정보가 어떻게 수집, 이용, 보호되는지를 안내합니다.

본 처리방침이 적용되는 앱
  • 주휴수당 계산기 — 시급/주 근무시간 입력 → 주휴수당 자동 계산
  • 4대보험 계산기 — 월급 입력 → 4대보험 + 소득세 + 실수령액 + 월별 저장 + 연말정산 예측 + 퇴직금 계산

2. 수집하는 개인정보 항목

2-1. 본 앱이 외부 서버로 수집하는 정보

없습니다.

본 앱은 사용자의 입력 데이터(시급, 근무시간, 월급, 입사일 등)를 외부 서버로 전송하거나 저장하지 않습니다. 사용자의 이름, 이메일, 전화번호, 위치 등 어떠한 개인 식별 정보도 수집하지 않습니다.

2-2. 4대보험 계산기 - 단말기 내부 저장

4대보험 계산기는 사용자 편의를 위해 다음 정보를 사용자의 단말기 내부 저장소(SQLite)에만 저장합니다. 외부 서버 전송 없음.

  • 월별 급여 기록 — 사용자가 명시적으로 "저장" 버튼을 눌렀을 때만 저장
    • 월 총 급여, 4대보험 차감액, 소득세, 실수령액
    • 산재 업종, 부양가족 수, 사업장 규모
  • 입사일 — 퇴직금 계산 시 사용자가 입력한 입사일

위 데이터는 사용자가 앱을 삭제하거나 "더보기 → 데이터 관리 → 전체 기록 삭제"를 선택하면 즉시 영구 삭제됩니다.

2-3. 광고 서비스(Google AdMob)가 수집하는 정보

두 앱 모두 광고 표시를 위해 Google AdMob을 사용합니다. AdMob은 다음 정보를 수집할 수 있습니다.

  • 광고 식별자(Advertising ID): 맞춤 광고 게재 및 광고 효과 측정
  • 기기 정보: 기기 모델, OS 버전, 언어 설정
  • IP 주소: 대략적인 위치 추정 및 사기 방지
  • 광고 상호작용: 광고 노출, 클릭, 시청 시간

자세한 내용은 Google 개인정보 처리방침을 참고하시기 바랍니다.

3. 개인정보의 이용 목적

본 앱은 사용자 개인정보를 직접 수집하지 않으나, 광고 서비스 운영을 위해 다음 목적으로 이용됩니다.

항목이용 목적
광고 식별자맞춤 광고 게재, 광고 빈도 제한
기기 정보광고 호환성 확인, 사기 방지
광고 상호작용광고주에게 보고할 통계 산출

4. 개인정보의 보유 및 이용 기간

본 앱은 사용자 정보를 외부 서버에 보유하지 않습니다.

  • 4대보험 계산기 단말기 저장 데이터: 사용자가 명시적으로 삭제하거나 앱 삭제 시 영구 삭제
  • AdMob 수집 정보: Google의 보유 정책에 따라 처리

사용자는 언제든지 다음 방법으로 광고 식별자를 재설정할 수 있습니다.

  • Android: 설정 → Google → 광고 → 광고 ID 재설정 / 광고 ID 삭제

5. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 광고 서비스 운영을 위해 다음 사업자와 정보가 공유될 수 있습니다.

제공받는 자제공 정보이용 목적
Google LLC (AdMob)광고 식별자, 기기 정보, IP광고 게재 및 측정

6. 개인정보의 안전성 확보 조치

  • 사용자 입력 데이터는 단말기 외부로 전송되지 않음
  • 4대보험 계산기 저장 데이터는 단말기 내부 저장소(앱 전용 영역)에만 보관
  • 광고 데이터 전송은 HTTPS 암호화 통신 사용
  • AdMob의 보안 정책에 따른 데이터 처리

7. 사용자의 권리

  1. 저장 데이터 삭제: 4대보험 계산기 → 더보기 → 데이터 관리 → 전체 기록 삭제
  2. 광고 ID 재설정/삭제: 단말기 설정에서 직접 가능
  3. 맞춤 광고 거부: Android 설정 → Google → 광고 → "광고 맞춤설정 사용 중지"
  4. 앱 삭제: 본 앱을 삭제하면 모든 데이터 수집/저장이 중단되고 단말기 저장 데이터도 영구 삭제됨

8. PDF 출력 기능 (4대보험 계산기)

4대보험 계산기의 PDF 명세서 출력 기능은 Android 시스템 인쇄 API를 사용하며, 다음 사항을 안내드립니다.

  • PDF는 단말기에서 생성되며 외부 서버로 전송되지 않습니다.
  • 사용자가 시스템 다이얼로그에서 "PDF로 저장", "Drive에 저장", "Gmail" 등을 선택할 수 있습니다.
  • 저장 위치 및 공유는 사용자의 명시적 선택에 의해서만 이루어집니다.

9. 아동의 개인정보 보호
